# Myth Fu

**Type**: Unkillable · **Difficulty**: Expert · **Key**: 1 Key UNM · **Affinity**: All Affinities · **Author**: Saphyrra

Source: https://deadwoodjedi.com/speed-tunes/myth-fu/

Myth-Fu is a highly recommended, full-auto speedtune with a high speed 3:1 Demytha while Heiress cleanses the debuffs to make it affinity friendly.

## Slot Rules

### Slot 1 · Demytha

Can go faster but other champions speeds need to be adjusted.

### Slot 2 · Heiress

Affinity friendly if: Heiress > 282.90 true speed

### Slot 4 · Seeker

Affinity friendly if: DPS < Seeker true speed * (7.15/7.8)

### Slot 5 · DPS

Affinity friendly if: DPS < Seeker true speed * (7.15/7.8)

## Notes

The numbers tend to have very little wiggle room. Changing the speeds of one champion also often necessitate changing the speeds of another as they are intertwined. Double check everything in the calculator before making any alterations to these speeds. Using a champion with a speed aura lead is heavily recommended (Fu-Shan seems to be the best choice). Listed Speeds INCLUDE the aura.


TUNE CAN FAIL AGAINST SPIRIT IF THE NOTES BELOW ARE NOT FOLLOWED, EVEN IF SHOWN AS SYNCING IN THE CALCULATOR!


Notes for Spirit Affinity:



Heiress > 282.90 true speed
DPS < Seeker true speed * (7.15/7.8) (Tip: (7.15/7.8) = 0.916666666666667)

The “True Speed” is displayed in the calculator below the speed that you input. This is the speed the tune is using.

Aura/Speed Sets/LOS Mastery can effect the true speed so make sure you input these!


Example True Speeds from DPS Link below:

https://deadwoodjedi.info/cb/b6c02283176ed571abb213e879b6dd974e123aac

Heiress = 283.83

Seeker = 208.72

Warboy = 190.84

———————–

So Warboy True Speed needs to be LESS then seekers true speed 208.72 * 0.916666666666667 = 191.326666666666736

→ Warboy needs to be under 191.33 true speed shown in the calc.
