# Fast Tower Skullcrusher Unkillable

**Type**: Unkillable · **Difficulty**: Moderate · **Key**: 3 Key UNM · **Affinity**: All Affinities · **Author**: DeadwoodJedi

Unkillable team using Roschard The Tower to block damage on the clan boss AOE hits and skullcrusher to tank the stun with his self unkillable.

## Slot Rules

### Slot 1 · Skullcrusher

For the Clan Boss to target Skullcrusher he needs to have about 16% of his max HP remaining minimum.


Using an immortal set and life drinker mastery help maintain that HP level.

### Slot 5 · Block Debuffer

Can replace this champ with an AOE 3 turn cooldown block debuffs champ to make it spirit friendly.

Speed Tune Sync:



UNM tune will sync on turn 6

Skill Set Up:



UNM

Skullcrusher: Open with A2
Tower: Open with A3


NM

Skullcrusher: Open with A2

On Turn 8 ( After the 3rd AOE2 ) add in a 1 turn delay on Skullcrushers A2 so he uses it immediately before the stun.


Tower: Open with A3

On Turn 9 ( After the 3rd stun ) add in a 1 turn delay on Towers A3 so he uses it immediately before the AOE1.





Gear, Masteries, Blessings:



Gear

Any sets that don’t affect turn meter or skill timing for the key players (Skullcrusher/Tower).
No relentless.


Masteries

Stick to typical Masteries that won’t affect Turn Meter or skill timing.
No Deterrence on Skullcrusher


Blessings

Stick to typical Clan Boss Blessings.  ( 1 Brimstone, 1 Cruelty, Remainder Phantom Touch )



Advanced Notes:



The trick is getting the stun to land on Skullcrusher. He must have SOME HP or the CB won’t target him – easiest way is to give him 1 set of Immortal gear, though it’s possible without by using the Lifedrinker Mastery and having high crit rate and damage.
Affinity Adjustments

Tower/Skullcrusher Affinity Guide

Magic Affinity

Swap Skullcrusher with Gnarlhorn


Spirit

Use a champ with AOE Block Debuff Skills on a 3 turn Cooldown in the 191 speed slot.


Force

Run a Magic Affinity DPS higher in the team order than Tower to pull the stun from him.
Adjust HP/DEF values of magic DPS and Skullcrusher so that eventually Clan Boss will target Skullcrusher for the stun as he is “Killable”.
