# Myth Heir

**Type**: Unkillable · **Difficulty**: Hard · **Key**: 1 Key UNM · **Affinity**: All Affinities · **Author**: Facemelter

Source: https://deadwoodjedi.com/speed-tunes/myth-heir/

A full-auto affinity friendly team that uses Demytha's block damage to stay unkillable. Deacon and Seeker push turn meter to keep speeds low, and Heiress cleanses debuffs while bringing a speed buff.

Achieving 1 key UNM with this team requires high end gear.

## Slot Rules

### Slot 1 · Demytha

Must set A2 as priority 2 and A3 as priority 1

### Slot 2 · Heiress

Must set A2 as priority 1

Heiress true speed > (7 / 5.05) * Seeker true speed (fraction approx. 1.39)

### Slot 3 · Deacon

Must set A1 as opening skill, A2 as priority 1, and A3 as priority 2

### Slot 4 · Seeker

Must set A1 as opening skill

### Slot 5 · DPS

DPS true speed < (7.15 / 7.8) * Seeker true speed (fraction is approx. 0.916)
