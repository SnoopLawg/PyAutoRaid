# Batman Forever

**Type**: Unkillable · **Difficulty**: Hard · **Key**: 1 Key UNM · **Affinity**: All Affinities · **Author**: Napoleon Camembert

Source: https://deadwoodjedi.com/speed-tunes/batman-forever/

This is the unkillable version of the Batman 2:1 with Seeker.  Can be made full auto and affinity friendly with a cleanser or block debuffs champion.

## Slot Rules

### Slot 3 · DPS

Can use a 3 turn block debuff champion to prevent either the stun or affinity debuffs, but not both.

### Slot 5 · DPS/Cleanser

To stay affinity friendly a doompriest should occupy this slot to cleanse both stun and debuffs every time.  If not against Spirit affinity (or willing to tank the stun) any 3 turn cleanse can be used to deal with the stun or affinity debuff, but not both.

Gear, Masteries, Blessings:



Gear

Any sets that don’t affect turn meter or skill timing for the key players (Seeker/4 Turn UK).
Relentless okay for DPS.


Masteries

Stick to typical Masteries that won’t affect Turn Meter or skill timing.


Blessings

Stick to typical Clan Boss Blessings.  ( 1 Brimstone, 1 Cruelty, Remainder Phantom Touch )



Advanced Notes:



If running 2 DPS one of them must take the stun.

Its helpful to run the UK champs and Seeker in one set of Immortal to heal back poison damage taken against Void.


Ninja Notes

Must open with A3.
If using combined with an Ally Attack Champion

Must prioritize his A3 as 1st choice and A2 as 2nd choice.
Cannot use Relentless or Cycle of Magic or Counter-Attack Masteries or Refresh or Counter-Attack Accessories.
Ally Attack Champion must both prioritize that skill and open with it – this is generally their default.


If NOT using an Ally Attack Champion

Leave his skill priorities to default (still must open with A3) and use Refresh or Counter-Attack Masteries and Accessories or Relentless.
